# 📈 Stock Market Analysis with Polygon.io

This is a Streamlit app that fetches and visualizes historical stock data using the [Polygon.io API](https://polygon.io).  
Built for analysts, traders, or anyone interested in time series visualization.

---

## 🖼️ App Screenshot

![App Screenshot](https://user-images.githubusercontent.com/your-screenshot-url)

---

## 🔍 Features

- 🔎 Real-time stock data via Polygon.io
- 📅 Custom date range input
- 📊 Line chart with 20-day & 100-day moving averages
- ☁️ Hosted live on Streamlit Cloud
